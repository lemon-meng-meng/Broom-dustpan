## 安装说明

1. 安装 Tampermonkey 或 Violentmonkey 扩展
2. 获取免费 API Key：https://ocr.space/ocrapi/freekey
3. 点击Tampermonkey图标 → 设置 OCR API Key

## 使用方法

1. 截图（Win+Shift+S 或 Cmd+Shift+4）
2. 按 Alt+Q 或点击Tampermonkey菜单
3. 确认后自动识别并复制文字

## 安装
安装 Tampermonkey 拖入 Broom-dustpan.js 安装脚本并替换为你在 OCR.space 申请的免费 API Key。
