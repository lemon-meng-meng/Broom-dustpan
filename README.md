# Broom-dustpan
浏览器内截图后按 **Alt+Q** 一键 OCR（基于 OCR.space）。

## 安装
1. 安装 [Tampermonkey](https://www.tampermonkey.net/)  
2. 点击 [Broom-dustpan.js](./Broom-dustpan.js) 安装脚本  
3. 将 `YOUR_API_KEY_HERE` 替换为你在 [OCR.space](https://ocr.space) 申请的免费 API Key。
