## 安装说明
1. 安装 Tampermonkey 或 Violentmonkey 扩展
2. 点击 [此处](https://raw.githubusercontent.com/lemon-meng-meng/Broom-dustpan/main/OCR%E8%84%9A%E6%9C%AC%EF%BC%88%E4%B8%AA%E4%BA%BA%E7%94%A8%EF%BC%89.js) 安装脚本
3. 获取免费 API Key：https://ocr.space/ocrapi/freekey
4. 点击Tampermonkey图标 → 设置 OCR API Key

## 使用方法
1. 截图（Win+Shift+S 或 Cmd+Shift+4）
2. 按 Alt+Q 或点击Tampermonkey菜单
3. 确认后自动识别并复制文字
